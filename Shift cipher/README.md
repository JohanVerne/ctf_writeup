# Shift cipher

## Challenge

Recover the password
Index : keep on turning.

## Solution

L'indice + le nom du challenge me laissent penser que le fichier est encodé avec un chiffrement par rotation.
On peut le décrypter automatiquement sur le site https://www.dcode.fr/chiffre-rot

Le plain text était chiffré avec un ROT ASCII+10
