# HTTP - User-agent

## Challenge

Admin is really dumb...

## Solution

En accédant au site, on obtient une erreur que le user-agent n'est pas admin.
Le créateur du site nous donne une réponse directe pour obtenir le MDP.

Avec ThunderClient, on modifie notre user-agent en 'admin' et on fait une requête au site pour obtenir le mot de passe
